This application will display data, stored in SQL database "AutoSpareParts" in a window.

Application is designed for Windows.
You will have some sorting options for car spare parts in this application.

For application program work you need:
SQL database "AutoSpareParts" up and running with access to it;
Proper connection string provided via file ConnectionString.txt in application root directory;

If you want to assemble the application by yourself you will need a .NET compiler, which is included in MS Visual Studio.
